About:

Secure, support and speed up the installation of non-metallic, service entrance, metal clad and data/communication cables with the Wire Bracket Device. The Wire bracket device installation allows cables to be installed in a neat, orderly and safe manner. Save time by creating a custom and orderly raceway. The device also saves the buildingâ€™s structure by reducing or eliminating the need to bore holes for electrical cables.

Residential Use:

Install cable in a custom raceway (of any distance) in open ceilings, dropped down ceilings, attics, furred down areas, crawl spaces, open basements and equipment back boards. Current residential wiring installation allows three cables to be installed in each bored hole from one end of the structure to the other. Use of the wire bracket device significantly reduces structural damage to wood joists by limiting the number of holes bored through the joists in order to install cable. Installation of the wire bracket device reduces the number of staples installed. Installing fewer staples reduces the possibility of increased resistance when staples are installed too tight.

Commercial Use:

Install MC cable in a custom raceway (of any distance) by attaching the wire bracket device to concrete, aluminum or steel. Install up to ten 12/2 with ground cables (see capacity guide) in a single device. Custom built raceways eliminate the need to follow an existing path that is not necessarily the shortest and easiest route to follow.

Data/Communication Use:

Install Data/Communication cables in a custom raceway (of any distance) as listed in the residential and commercial use areas above. Low voltage raceways can easily be built away from electrical fields and cables can be installed in a neat, orderly and safe manner.
